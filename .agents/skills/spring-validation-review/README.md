# Spring Validation Review Skill

用于审查和补全 Spring Boot 项目中的 Bean Validation：DTO 字段约束、Controller 的 `@Valid`、嵌套对象级联校验、直接方法参数约束，以及 Service 业务校验边界。

## 安装到当前仓库

将整个 `spring-validation-review` 目录复制到：

`.agents/skills/spring-validation-review/`

## 安装为个人 Skill

将目录复制到：

`$HOME/.agents/skills/spring-validation-review/`

## 调用示例

`$spring-validation-review 检查本次优惠券接口改动，补全缺失校验并运行相关测试。`

也可以直接说：

`检查这些 DTO、Controller 和 Service 是否缺少参数校验，按项目规范修复。`
