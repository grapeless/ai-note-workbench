## 介绍  

本地化的Rag系统，React + SpringBoot，通过web端界面交互，将原数据处理后持久化在本地。

- 相比于云Rag，本地存储，无需支付线上存储费用
- 相比于直接丢给网页AI/本地Agent，为持久化的Rag，拥有持久化和Rag相比于直接对话的所有优点

## 功能

- [x] **文件上传与解析**，由于是该应用是本地工具向的，除了需要将原文件处理后变为向量存入数据库，由于前端框架限制，为了方便编辑功能实现，需要在服务器/本地保存副本方便处理。
- [ ] **AI基础对话，支持多模型切换**
- [ ] **支持工具调用，MCP**
- [ ] **在线编辑md/txt，实时预览pdf/word**
- [ ] **支持RAG**
- [ ] **RAG引用跳转 + 原文高亮**

- [ ] Agent能力，能让AI编辑所选文件
- [ ] 总结页面，数据可视化页面
